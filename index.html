Giphy-Ajax Prototype

 expand_less
Github Page
Giphy-Ajax Prototypeexpand_more

My first project built to utilize Ajax to use an API to pull data and append it to the DOM dynamically. Has a pre-defined list of 'tags' to search Giphy's API with, displays up to 10 appropriate gifs on the page. The gifs start as still images, but clicking them will turn them from still to playing, and vice versa. A user can add their own tags, though these are not saved between sessions.

Built using Javascript, HTML, and CSS. Utilizes jQuery, Ajax, and Bootstrap. Giphy API utilized for non-commercial use. I take no responsibility for any of the content of the gifs pulled by the API.
